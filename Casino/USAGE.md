# Казино — руководство пользователя

## Запуск

```bash
dotnet run --project Casino.csproj
```

### Поток игры

1. Программа просит ввести начальный баланс. Допустимы целые и дробные положительные числа, разделитель — точка (100,
   250.50). Отрицательное или некорректное значение запрашивается заново.

### Пункты меню

1. Начать — печатает текущий баланс и подтверждает старт игры.
2. Показать баланс — выводит текущий баланс.
3. Сыграть раунд — запрашивает ставку, разыгрывает раунд и показывает результат.
4. Выйти — завершает программу.

### Ограничения на ставку

1. Ставка должна быть положительным числом.
2. Ставка не должна превышать текущий баланс.
3. Если баланс равен 0, опция «Сыграть раунд» сообщит, что ставку сделать нельзя.
