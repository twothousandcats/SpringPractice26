# CarFactory — руководство пользователя

## Запуск

```bash
dotnet run --project CarFactory.csproj
```

## Информация

Программа представляет собой конструктор создания машины

1. Пользователь должен ввести имя бренда производителя
2. А далее выбрать предложенную кофигурации
3. В конце выводится саммари конфига созданной машины
