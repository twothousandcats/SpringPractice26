# Fighters

Консольная игра-арена: пользователь собирает бойцов из расы, класса, оружия и брони, после чего натравливает их друг на
друга в пораундовой битве. Победитель — последний оставшийся в живых

## Консольные команды (Имена команд регистронезависимы)

1. Add - Пошагово собрать бойца (имя, раса, класс, оружие, броня) и добавить на арену
2. List - Показать бойцов на арене с их характеристиками и текущим HP
3. Remove - Удалить бойца по индексу
4. Play - Запустить битву (от 2 бойцов). После боя арена очищается
5. Help - Показать список команд
6. Exit - Выйти
